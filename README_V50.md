# Excel AI Dịch Việt - Nhật - Release V50

Đây là gói đưa lên GitHub cho phiên bản V50 của app dịch Excel AI Việt - Nhật.

## Phiên bản

- Release: V50
- File chạy chính: `dist/Excel_AI_Dich_Viet_Nhat_v50_compact_company_card.exe`
- Spec build tương ứng: `Excel_AI_Dich_Viet_Nhat_v50_compact_company_card.spec`

## Chức năng chính

- Chọn file Excel và dịch nội dung tiếng Việt sang tiếng Nhật.
- Xuất file Excel mới có hậu tố `_JP`.
- Giữ cấu trúc/bố cục Excel nhiều nhất có thể.
- Có giao diện công ty và thông tin app theo bản V50.

## Lưu ý bảo mật

- Không đưa API Key cá nhân lên GitHub.
- Người dùng phải tự nhập API Key trên máy của mình.
- File cấu hình local và API key đã được loại khỏi Git bằng `.gitignore`.

## Cách dùng nhanh

1. Tải file `.exe` bản V50.
2. Mở app.
3. Chọn file Excel cần dịch.
4. Nhập API Key.
5. Bấm bắt đầu dịch và xuất file `_JP`.

## Ghi chú

Bản V50 hiện còn dưới dạng file phát hành `.exe` và file `.spec`. Source `app.py` trong thư mục làm việc hiện tại đã được phát triển tiếp lên các bản mới hơn, nên repo này đánh dấu rõ V50 như một release riêng.
