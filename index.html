Trang tải app desktop Windows dùng để dịch file Excel tiếng Việt sang tiếng Nhật và xuất file _JP. Đây là bản phát hành V50. Windows用デスクトップアプリのダウンロードページです。Excelのベトナム語を日本語へ翻訳し、_JPファイルとして出力します。V50版です。

Cách dùng nhanh 簡単な使い方

  • Tải file .exe. / .exeファイルをダウンロードします。
  • Mở app trên Windows. / Windowsでアプリを開きます。
  • Chọn file Excel cần dịch. / 翻訳したいExcelファイルを選択します。
  • Nhập API Key. / APIキーを入力します。
  • Bấm dịch và lấy file _JP. / 翻訳を実行し、_JPファイルを取得します。

Lưu ý 注意事項

  • Render chỉ dùng làm trang tải app, không chạy app Excel trực tiếp trên web. / Renderはアプリのダウンロードページ用であり、Web上でExcelアプリを直接実行するものではありません。
  • App desktop cần chạy trên máy Windows có Excel/WPS phù hợp. / デスクトップアプリは、対応するExcelまたはWPSが入ったWindows PCで実行してください。
  • Không đưa API Key cá nhân lên GitHub hoặc Render. / 個人のAPIキーをGitHubやRenderへアップロードしないでください。
