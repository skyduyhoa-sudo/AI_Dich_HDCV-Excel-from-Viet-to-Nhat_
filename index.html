Excel AI Dịch Việt - Nhật V50 Excel AI ベトナム語・日本語翻訳

Trang tải app desktop Windows dùng để dịch file Excel tiếng Việt sang tiếng Nhật và xuất file _JP. Đây là bản phát hành V50.

Tải app

Bấm nút dưới để tải file chạy Windows:

⬇ Tải Excel AI Dịch Việt - Nhật V50

Nếu trình duyệt cảnh báo file .exe, hãy chọn giữ lại nếu anh tin tưởng nguồn tải nội bộ này.

Cách dùng nhanh

  • Tải file .exe.
  • Mở app trên Windows.
  • Chọn file Excel cần dịch.
  • Nhập API Key.
  • Bấm dịch và lấy file _JP.

Lưu ý

  • Vercel chỉ dùng làm trang tải app, không chạy app Excel trực tiếp trên web.
  • App desktop cần chạy trên máy Windows có Excel/WPS phù hợp.
  • Không đưa API Key cá nhân lên GitHub hoặc Vercel.
